## 🆕 新功能：背单词学习系统

本博客新增了完整的背单词学习功能，帮助考研备考！

### ✨ 功能特性

- **📊 学习进度追踪**：侧栏实时显示今日进度和本周统计
- **📖 每日一词**：每天自动推送一个新单词
- **🎴 卡片式学习**：翻转卡片查看释义，沉浸式学习体验
- **⌨️ 快捷键支持**：← ↓ → 空格，高效操作
- **📚 多词库切换**：考研、四级词汇随心选
- **💾 数据持久化**：学习进度自动保存，刷新不丢失

### 🚀 快速开始

```bash
# 构建网站
python build.py build

# 启动预览
python build.py preview

# 访问背单词页面
http://localhost:3000/Tools/Vocabulary
```
