# 📚 词汇复习功能演示指南

## 🎯 快速体验复习功能

### 方法一：自动生成模拟数据（推荐）

1. **清除现有数据**（如果有的话）
   - 打开浏览器控制台（F12）
   - 输入：`vocabDevTools.clearAllData()`

2. **刷新页面**
   - 系统会自动生成 15 个"昨天学过的单词"
   - 其中约 10-11 个需要今天复习

3. **查看效果**
   - 侧栏会显示：🔄 待复习 X 词
   - 单词卡片上方会显示：🔄 复习 或 📖 新词

打开浏览器控制台（F12），可以使用以下命令：

```javascript
vocabDevTools.showProgress()

// 查看今日统计
vocabDevTools.showStats()

// 模拟昨天学了 10 个词
vocabDevTools.simulateLearn(10)
```

- **不认识的词**：昨天学 → 今天复习
- **模糊的词**：昨天学 → 今天复习
- **认识的词**：昨天学 → 3天后复习

所以今天应该有 **10-11 个词需要复习**！

## 🎓 复习算法说明

### 间隔时间表

| 掌握程度 | 第1次 | 第2次 | 第3次 | 第4次 | 第5次 | 第6次 |
|---------|------|------|------|------|------|------|
| 不认识   | 1天  | 1天  | 1天  | 1天  | 1天  | 1天  |
| 模糊     | 1天  | 2天  | 4天  | 7天  | 15天 | 15天 |
| 认识     | 3天  | 7天  | 15天 | 30天 | 60天 | 90天 |

### 学习策略

1. **不认识**：每天复习，直到掌握
2. **模糊**：逐渐拉长间隔，巩固记忆
3. **认识**：长间隔复习，保持记忆

## 🚀 使用建议

### 最佳实践

1. **每天坚持**：即使只学几个词也比不学好
2. **诚实评估**：根据真实掌握程度选择
3. **优先复习**：先完成复习任务
4. **适量学习**：建议每天 20-30 词

### 学习节奏

```
Week 1: 每天 20 新词
Week 2: 复习 + 新词（约 15 复习 + 5 新词）
Week 3: 稳定期（约 10 复习 + 10 新词）
Week 4+: 长期记忆形成
```

## ❓ 常见问题

### Q1: 为什么没有看到复习提示？

**A:** 可能原因：
1. 首次使用，还没有学过单词
2. 复习时间未到
3. 浏览器缓存问题

**解决方法：**
```javascript
// 重新生成模拟数据
vocabDevTools.regenerateMockData()
```

### Q2: 如何清除所有数据重新开始？

**A:** 
```javascript
vocabDevTools.clearAllData()
```

### Q3: 如何查看当前有多少词需要复习？

**A:** 
- 查看侧栏的"待复习"卡片
- 或在控制台执行：`vocabDevTools.showProgress()`

### Q4: 模拟数据会影响真实学习吗？

**A:** 
- 模拟数据只在首次使用时生成
- 如果已有学习数据，不会生成模拟数据
- 可以随时清除：`vocabDevTools.clearAllData()`

### Q5: 如何测试不同的复习场景？

**A:**
```javascript
// 模拟昨天学了 5 个词
vocabDevTools.simulateLearn(5)

// 模拟昨天学了 20 个词
vocabDevTools.simulateLearn(20)
```
